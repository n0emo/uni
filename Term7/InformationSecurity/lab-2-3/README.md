# Лабораторная работа 2.3

Сервер для обмена зашифрованными сообщениями. После подключения к серверу,
клиент будет получать сообщения от всех остальных подключенных клиентов.
Сообщения передаются в зашифрованном виде, при этом сервер имеет доступ к
открытым сообщениям.

## Быстрый старт

Запустить сервер:

```console
go run server/main.go 127.0.0.1:8080
```

Запустить клиент:

```console
go run client/main.go 127.0.0.1:8080
```
